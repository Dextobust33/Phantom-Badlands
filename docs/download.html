How to Get Started

  1. Download the launcher ZIP above
  2. Extract to any folder
  3. Run PhantomBadlandsLauncher.exe — it downloads the latest game client automatically
  4. Create an account and your first character
  5. Type /help in-game for a full guide

Direct Download (No Launcher)

Prefer to skip the launcher? Download the client directly from GitHub Releases.

System Requirements

  • OS: Windows 10/11 (64-bit)
  • RAM: 512 MB
  • Storage: 50 MB
  • Network: Internet connection required
  • GPU: Any (text-based rendering)
