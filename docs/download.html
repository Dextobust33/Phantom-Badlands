How to Get Started

  1. Download the latest release ZIP from GitHub
  2. Extract the ZIP to any folder
  3. Run PhantomBadlandsClient.exe
  4. Create an account and your first character
  5. Type /help in-game for a full guide

System Requirements

  • OS: Windows 10/11 (64-bit)
  • RAM: 512 MB
  • Storage: 50 MB
  • Network: Internet connection required
  • GPU: Any (text-based rendering)
