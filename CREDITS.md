## Character Sprites (LPC — Liberated Pixel Cup)

Player character sprites are composed from the **Liberated Pixel Cup**
asset library via the LPC Universal Sprite Sheet Character Generator.
The pack draws on many community contributors; primary attribution
goes to wulax (LPC Medieval Fantasy Character Sprites) and the
generator project itself.

- **"LPC Medieval Fantasy Character Sprites"** by **wulax**
  - Source: <https://opengameart.org/content/lpc-medieval-fantasy-character-sprites>
  - License chosen for this project: **OGA-BY 3.0** (<https://static.opengameart.org/OGA-BY-3.0.txt>) — author triple-licenses under CC-BY-SA 3.0 / GPL 3.0 / OGA-BY 3.0; we select OGA-BY 3.0 to avoid share-alike obligations.
  - Modifications: assembled, layered, and recolored via the LPC Character Generator.

- **LPC Universal Sprite Sheet Character Generator**
  - Web tool: <https://liberatedpixelcup.github.io/Universal-LPC-Spritesheet-Character-Generator/>
  - Repository: <https://github.com/LiberatedPixelCup/Universal-LPC-Spritesheet-Character-Generator>

Per-part attribution for every part used in every class is preserved
verbatim in `client/sprites/classes/<class>.credits.txt`. Those files
ship with the release (do not delete them — they are the OGA-BY 3.0
paper trail). The full per-part credits are inlined below for
convenience.
